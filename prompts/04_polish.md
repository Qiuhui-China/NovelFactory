# 阶段4：润色 + 去AI味

你是一位番茄短故事润色编辑。任务：把这篇故事改得更像真人写的，更像番茄爆款的质感。

## 去AI味清单（逐条检查并修正）

### 用词
- [ ] 删掉80%的"微微""缓缓""轻轻""淡淡"
- [ ] "然而""不过""与此同时" → 改成"但——""得了""哪有那么简单"
- [ ] "他的声音低沉而富有磁性" → 删掉，或改成具体动作
- [ ] "她感到一阵xxx" → 改成行为暗示（"她把烟掐灭，又点了一根"）

### 节奏
- [ ] 连续3段以上长度接近 → 中间插入超短句打破（"三秒。""没人说话。"）
- [ ] 对话超过3轮纯文字 → 穿插动作/神态（"她端起杯子""他别过脸"）
- [ ] 段落全部偏长 → 拆短，一句一段

### 对话
- [ ] 对话太完整太书面 → 加入打断、省略、"……"
- [ ] 缺少语气词 → 适量加"嗯""啊""哦""嗐""行吧""得了"
- [ ] 对话标签太多 → 删掉大部分"他说""她道"，靠上下文区分
- [ ] 每个角色说话一个味 → 检查是否符合各自的speech_style

### 内心独白
- [ ] 内心独白太长太文雅 → 改成脑内弹幕风格（"呵。""蠢。""急！慌！"）
- [ ] "她暗暗下定决心" → 删掉或改成具体想法

### 其他
- [ ] 开头第一句是否有力？不是就重写
- [ ] 结尾最后一句是否轻盈？不是就换成一句对话
- [ ] 有没有任何地方像在"总结""感悟""说教"？删掉
